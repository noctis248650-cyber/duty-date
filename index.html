📖 Firebase 무료 설정 가이드 (5분)
1. console.firebase.google.com 접속 (Google 로그인)
2. "프로젝트 추가" → 이름 입력 (예: duty-date) → Analytics 비활성화 → 만들기
3. 좌측 메뉴 "빌드 → Realtime Database" → "데이터베이스 만들기"
4. 위치: asia-southeast1 선택 (한국 가까움)
5. 보안 규칙: "테스트 모드에서 시작" 선택
6. 데이터베이스 페이지 상단 URL 복사 (https://...firebaseio.com 또는 https://...firebasedatabase.app)
7. 위 입력칸에 붙여넣고 "연결" 버튼
8. 같은 URL을 여자친구분 폰에서도 동일하게 입력 → 자동 동기화 시작! 💕
